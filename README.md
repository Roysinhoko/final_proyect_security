# SeniorSafeWeb – Seguridad digital para personas mayores

## Idea Principal del Proyecto
> Crear una aplicación educativa hecha en Python que ayude a las personas mayores a entender mejor la tecnología y reconocer estafas digitales mediante ejemplos simples, simulaciones y herramientas interactivas.

## Objetivos
- Explicar tecnología en lenguaje simple  
- Enseñar a detectar estafas comunes  
- Simulaciones sin riesgo  
- Sistema interactivo que analiza mensajes sospechosos  
- Consejos rápidos y botón de ayuda  
- Soporte por voz usando pyttsx3  

## Tecnologías usadas (Python)
- Python  
- Flask (para la plataforma web)  
- Pyttsx3  
- SQLite  
- IA simple para detección de mensajes sospechosos  

## ¿Por qué este proyecto?
Las personas mayores son uno de los grupos más afectados por estafas digitales.  
SeniorSafeWeb busca educarlas, protegerlas y acompañarlas con herramientas creadas especialmente para ellas utilizando Python.
